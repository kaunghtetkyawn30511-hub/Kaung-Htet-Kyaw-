# Kaung Htet Kyaw Portfolio

Static GitHub Pages portfolio for Kaung Htet Kyaw, Graphic Designer / Creative Designer.

## Structure

- `index.html` - primary one-page portfolio.
- `about.html` - profile page kept for existing links.
- `projects.html` - project category index.
- `projects1.html`, `projects2.html`, `projects3.html` - image galleries for social media, logo, and poster design.
- `styles.css` - shared visual system using the mockup color tokens.
- `script.js` - reveal animation, desktop custom cursor, and gallery lightbox.
- `KaungHtetKyaw_CV.pdf` - CV download target.

## Local preview

```sh
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Deploy

This repository is plain static HTML/CSS/JS. GitHub Pages can serve it directly from the repository root.
